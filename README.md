# CampusHub
>南京大学2026春《软件工程与计算II》课程项目

## 项目简介
CampusHub旨在为大学生之间日常的互助需求提供一个**统一的校园互助服务平台**。其功能包括发布需求、发现匹配、建立信任、完成协作等。
